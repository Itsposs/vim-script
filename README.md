
### vim script and gdb script

#### cscope usage
```bash
1. find . -type f > cscope.filescscope
2. cscope -Rbq
3. :cs add scope.out
   :cs find s tagname
            g tagname
            d tagname
            c tagname
            t tagname
            f tagname
            i tagname
```
